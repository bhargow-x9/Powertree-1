PowerTree is a **small wind energy system** designed to collect power from **slow and changing wind**.

It uses **many small wind units** placed on a tree-like structure instead of one large turbine.  
This helps the system work better in places where wind direction and speed change often.

This repository contains **documentation and study material** for team members and students.
